# Common Logs

Use this folder for logs created while executing the shared sequential plan in [`docs/plans/common-sequential-plan.md`](../../plans/common-sequential-plan.md).

Log files in this folder should cover:

- shared foundation tasks
- cross-stream integration tasks
- final stabilization tasks

Prefer phase-oriented files such as `phase-1-foundation.md`, `phase-2-integration.md`, and `phase-3-stabilization.md`.
